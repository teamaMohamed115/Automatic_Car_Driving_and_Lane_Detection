# 4WD Robot Project - Technical Context

## System Environment
- **Device:** Raspberry Pi 4 Model B
- **OS:** Ubuntu 24.04 LTS (Noble Numbat)
- **Architecture:** ARM64
- **Camera:** Raspberry Pi Camera Module V2 (IMX219)
- **Key Fix:** Native drivers and apps were compiled from source to support Ubuntu 24.04.

## Camera Configuration (CRITICAL)
The standard camera stack was broken; we use a custom-built stack.
- **libcamera:** Installed from source.
- **rpicam-apps:** Installed from source with OpenCV enabled.
- **GStreamer Pipeline:**
  ```text
